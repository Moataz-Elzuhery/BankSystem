# 💳 Bank Management System (C++ | OOP)

A console-based Bank Management System developed using **C++** with a strong focus on **Object-Oriented Programming (OOP)** principles.
This project simulates real-world banking operations such as managing clients, transactions, and account data.

---

## 🚀 Features

* 🔐 User Authentication (Login System)
* 👤 Add / Update / Delete Clients
* 💰 Deposit & Withdraw Money
* 🔎 Search for Clients
* 📋 Display All Clients
* 💾 File Handling (Persistent Storage)
* 🧱 Clean OOP Design (Classes & Encapsulation)

---

## 🧠 Concepts Applied

This project demonstrates solid understanding of:

* Object-Oriented Programming (OOP)

  * Classes & Objects
  * Encapsulation
  * Abstraction
* File Handling (Read / Write)

* Input Validation

## ⚙️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/bank-system.git
```

2. Navigate to the project folder:

```
cd bank-system
```

3. Compile the project:

```
g++ main.cpp -o bank-system
```

4. Run the program:

```
./bank-system
```
